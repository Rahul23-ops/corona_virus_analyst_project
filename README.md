🦠 COVID-19 Data Analysis using Python
📌 Project Overview

This project performs an exploratory data analysis (EDA) on a global COVID-19 dataset using Python. The analysis focuses on understanding the spread of the coronavirus by examining confirmed cases, deaths, recoveries, mortality rates, recovery rates, and trends over time.

The project demonstrates data manipulation, aggregation, and visualization techniques using the Pandas library.

🎯 Objectives
Analyze worldwide COVID-19 statistics
Identify countries and provinces with the highest cases
Calculate mortality and recovery rates
Analyze active cases
Study global trends over time
Practice real-world data analysis using Python
