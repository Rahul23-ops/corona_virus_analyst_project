# 🦠 COVID-19 Data Analysis using Python

## 📌 Project Overview

This project analyzes the global COVID-19 dataset using **Python** and **Pandas** to extract meaningful insights about the pandemic. It covers data cleaning, exploratory data analysis (EDA), and visualization to understand worldwide confirmed cases, deaths, recoveries, active cases, mortality rates, and recovery rates.

This project demonstrates practical data analysis skills that are essential for aspiring Data Analysts.

---

## 🎯 Project Objectives

- Calculate total confirmed COVID-19 cases worldwide
- Calculate total deaths worldwide
- Calculate total recovered cases worldwide
- Count the total number of countries/regions
- Analyze confirmed case trends over time
- Analyze active case trends over time
- Find countries with the highest confirmed cases
- Find countries with the highest death counts
- Calculate mortality rates by country
- Calculate recovery rates by country
- Analyze fatality and recovery rates by province/state
- Study the relationship between confirmed and recovered cases
